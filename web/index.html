What is Foundry Local?

Foundry Local is an on-device AI runtime and SDK for integrating optimized models into desktop and client applications. Models, prompts, and responses stay on the machine during normal inference.

01

Offline inference

Run cached models without depending on a hosted model endpoint.

Private by design

Keep documents, prompts, and generated responses on the device.

Hardware aware

Use available CPU, GPU, or NPU execution providers automatically.

03

Python quickstart

Initialize the SDK, select a model alias, load the model, and obtain a local chat client.

Python 
from foundry_local_sdk import Configuration, FoundryLocalManager

FoundryLocalManager.initialize(
    Configuration(app_name="foundry_local_guide")
)
manager = FoundryLocalManager.instance
model = manager.catalog.get_model("qwen2.5-1.5b")
model.download()
model.load()
client = model.get_chat_client()
